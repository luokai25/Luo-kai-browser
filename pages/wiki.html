Introduction

Luo Browser is a revolutionary web browser designed for both humans and AI agents. It uses the custom luo:// protocol for navigation.

Quick Start

To get started with Luo Browser, simply type a luo:// address in the address bar. Try these:

• luo://home - Home page
• luo://search - Search
• luo://agents - AI Agents

Building Sites

Creating a new luo:// site is easy! Just add an HTML file to the pages/ directory and register it in the browser.

AI Agents

Luo Browser comes with 8 built-in AI agents ready to help you with coding, research, content creation, and more!

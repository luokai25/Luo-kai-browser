Welcome to Luo Docs 📚

Luo Browser is a revolutionary browser designed for both humans and AI agents. With the custom luo:// protocol, it creates a unified space for navigation.

Quick Start

To navigate in Luo Browser, simply type a luo:// address in the address bar:

luo://home       → Luo Home
luo://search     → Luo Search  
luo://mail       → Luo Mail
luo://chat       → Luo Chat
luo://docs       → Luo Docs
luo://ai         → AI Assistant
🤖 For AI Agents: AI agents can access Luo Browser via REST API or direct URL access at /api/{endpoint}

luo:// Protocol

The luo:// protocol is the heart of Luo Browser. Unlike traditional HTTP, it provides:

• Unified Access - Both humans and AI can use the same URLs
• Built-in AI Support - Native AI integration
• Secure Sandboxing - Safe execution environment
